# Al-Tha3lab Al-Saree3

### Translation: "The Quick Fox"

Access the app here: https://al-tha3lab-al-saree3.netlify.app/

This is an arabic typing game built to help arabic language learners get comfortable with their arabic digital keyboard on desktop and mobile. The game currently contains a daily challenge mode with updating prompts to try every day.

Soon to come will be curated sets of challenges for beginners, novices, and experts, as well as a diacritic mode.
